
# vue-cli4 模板配置

基于vue-cli4的配置模板

封装了vue-router、axios、全局指令、全局过滤器、全局loading、全局路由切换动画等等

封装层次清晰，文件内都有对应的注释，方便查看。

## 插件
目前集成了
- svg组件（封装的svg直接引入即可）
- cdn配置
- gzip压缩
- 图片压缩
- 打包分析
- 别名配置
- 去掉console.log及注释

等等...

## 三方组件
全局引入element-ui, 默认配置了国际化，全局loading

图片懒加载


## 编码规范
配置eslint、stylelint、prettier 等规范，以及详细的配置说明。
